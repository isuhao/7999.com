域名搜索引擎

作者:邢志宇
 域名搜索引擎（Domain Name Search Engine）是查找已注册域名的详细信息和设想的域名是否已经注册，或者获得相关可用域名信息的专用工具，主要为企业或个人设计和选择域名服务，一般用户也可以利用关键词搜索，查找相同或相关主题的网站。它们的功能大致相同，但查询手段和检索方法各有特色，数据更新及时，没有重复结果和死链接。
　

    1、Checkdomain.com 
    http://www.checkdomain.com/
    1996年8月推出域名实时搜索引擎，查询世界各国域名，显示域名注册者全称、国别、地址、联系方式、注册时间、更新日期、域名有效截止日期、域名地址服务器(DNS) 等详细信息。
    2、Domainator 
    http://www.e-gineer.com/domainator/
    在搜索框中输入域名或关键词，可以快速查询该域名和包含该关键词的域名与商标是否已被注册，提供注册者详细信息。
    3、EuroSubmit.com Domain Name Search 
    http://www.eurosubmit.com/search.html
    查询各国域名信息，还可以为选择与确定自己的域名提供帮助。
    4、Domain name registries around the world 
    http://www.norid.no/domenenavnbaser/domreg.html
    查询各国域名注册情况的网站与数据库链接指南，按域名后缀或国家后缀分别依网站性质和所属国家查询域名注册信息。
    5、DomainSurfer 
    http://www.domainsurfer.com/
    搜索包括查询词的已注册域名，提供简要信息和网站链接。
    6、Clickey 
    http://www.clickey.com/
    提供关键词搜索和域名分类查功能，可进行国家与关键词限定查询，或国家与分类限定查询。
    7、DNS411 
    http://dns411.com/
    顶级域名（TLD）搜索引擎，用于查找特定顶级域名相关信息和顶级域名调查。如查询的域名已被注册，显示其所有人、国家、地址、联系方式、注册时间、更新日期、域名有效截止日期、域名地址服务器(DNS)等详细信息，如尚无注册，则提供注册表单备用。可以从域名查询其网站IP地址，也可以从IP地址反查域名。
    8、Domain Search 
    http://www.domainsearch.com/
    输入域名关键词，可以进行顶级域名和扩展域名同时/选择搜索，在提示已注册域名的同时，最多可以提供60多个可用扩展域名。已注册域名显示详细信息，可用域名分别给出注册价格及其授权使用时间。
    9、TargetDomain
    http://www.targetdomain.com/
    全球域名搜索引擎，输入域名查询注册情况，同时提供不同后缀的可用域名及其年注册费用，推荐相同后缀的相似域名，当查询的已注册域名终止或被注册人取消成为可用域名后，为申请者提供回购注册（Backorder）服务。
    10、Namedroppers.com 
    http://www.namedroppers.com/
    专门查询.com、.edu、.net、.org域名的搜索引擎。输入域名关键词，可以进行任意匹配、精确匹配、按首词查询，或已注册域名、建议域名、二者同时等多项选择搜索，已注册域名提供相关信息和网站链接，可用域名提供注册信息。
    11、deleteddomains
    http://www.deleteddomains.com/search.php
    1998年6月推出的已删除域名搜索引擎，用于查询最近终止，或即将终止的域名信息。其今日删除域名、今日注册域名、删除域名统计信息更新及时，极具查询价值。
    12、中国互联网络信息中心 (CNNIC)WHOIS查询
    http://www.cnnic.com.cn/
    可以查询除 .edu.cn 之外的所有以.cn结尾的英文，可以查询所有中文域名和通用网址。
    13、域名查询
    http://www.onlinenic.com/chinese/chk_domain.html
    互联网名称与地址分配组织（ICANN）正式授权的域名注册机构OnlineNIC,Inc公司的中文版域名查询系统。可进行.com 、.net、.org、.tv、.cc、.info、.com.cn、.net.cn、.org.cn、.cn、.us、.biz、.中国、.公司、.网络等国际域名和中文国内域名查询。 
    14、域名批量查询系统
    http://www.911b.com/trade/checktools/200410/100.html
    批量查询域名系统，一次最多可查询1000个域名 
    15、站长公告网域名查询
    http://www.zzgg.net/yuming.asp
    包括删除域名综合查询、四位以内域名查询、特殊含义域名查询三个版块。
    16、无忧搜网过期域名查询系统
    http://so.5eo.com/do/domain.asp?eo=37028
    中文界面的已删除域名查询系统。
